SimSem

SimSem is a tool that attempts to assign a meaningful semantic category to a given string. It does so using machine learning techniques, large-scale lexical resources and state-of-the-art fast approximate string matching. If everything is working correctly you should find a live demo of the system below.
SimSem is distributed under a permissive license and the source code can be found in its GitHub repository.
